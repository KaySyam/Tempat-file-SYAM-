# Proyek Pac-Man Pemula

Repositori ini berisi game sederhana bergaya Pac-Man yang dibuat dengan Python dan Pygame.
Proyek ini ditujukan untuk pemula yang ingin belajar dasar pemrograman game dengan cara yang mudah dan menyenangkan.

## File

- `simplepacman.py` - file utama game
- `ujigamepacman.py` - pengujian unit untuk beberapa logika game

## Kebutuhan

- Python 3
- Pygame

Instal Pygame dengan:

```bash
pip install pygame
```

## Cara Menjalankan

Jalankan game utama dengan:

```bash
python simplepacman.py
```

## Kontrol

- Tombol panah - menggerakkan Pac-Man
- `R` - memulai ulang setelah game over atau menang
- `Q` - keluar dari game

## Catatan

Proyek ini adalah contoh sederhana untuk belajar dan bisa dikembangkan lagi
dengan fitur baru, grafis yang lebih bagus, atau AI musuh yang lebih pintar.
